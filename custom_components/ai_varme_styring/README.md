# AI Varme Styring

![AI Varme Styring logo](logo.png)

Local Home Assistant integration for AI-based heating control.

## Current version
`0.2.6`

## Main features
- OpenClaw as primary decision engine
- Optional fallback to Ollama or Gemini
- Structured room-first decisions
- Stable fallback logic for safe runtime behavior

## HACS install
1. Add custom repository in HACS:
   `https://github.com/MRDonnii/ai-varme-styring`
2. Select type `Integration`
3. Install and restart Home Assistant

## Integration path
`custom_components/ai_varme_styring/`
